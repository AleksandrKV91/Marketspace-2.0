##Блок KPI - 5 карточек
1. CTR - отчет по sku колонки AK-BB
2. CR В КОРЗИНУ - отчет по sku колонки BD-BH
3. CR В ЗАКАЗ - отчет по sku колонки BJ-BN
4. ДОЛЯ РЕКЛ. ЗАКАЗОВ - отчет по sku колонки CC-CG
5. (CPO) Стоимость заказа = Маркетинговые расходы / Количество заказов.
