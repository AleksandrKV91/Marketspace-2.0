## Порядок работы при старте

1. Создать репозиторий `Marketspace-2.0` на GitHub
2. Инициализировать Next.js проект в папке `~/Desktop/marketspace-2.0/`
3. Создать таблицы в Supabase (см. 01-database.md)
4. Написать парсеры (02-parsers.md)
5. Написать API routes (03-api.md)
6. Реализовать фронтенд (04-frontend.md)
7. Загрузить данные: Свод → АВС → Потребность Китай → Остатки → Отчёт по SKU

## Исходные Excel файлы

В папке `Таблицы/`:
- `Свод.xlsb` — справочник SKU
- `АВС анализ февраль.xlsx` — АВС анализ
- `Потребность Китай.xlsx` — план поставок
- `Таблица Остатков.xlsx` — остатки и продажи
- `Отчет по sku.xlsb` — рекламные метрики
- `Основные показатели.xlsx` — справочник что откуда брать
